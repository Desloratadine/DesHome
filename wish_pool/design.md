# 弹球游戏 - 设计文档

## 1. 概述

一款基于物理弹球机制的html游戏。玩家操控底部横向球拍，在倒计时内尽可能多地接住从四周墙壁弹出的弹球来获取分数。

---

## 2. 核心玩法

### 游戏场景

- 游戏区域由 **上、左、右** 三面弹性墙壁围成，底部开放
- 球拍位于底部中央，可通过鼠标左右移动
- 球拍仅在水平方向移动，无法跳跃或上下移动

### 弹球生成

- 系统按固定时间间隔，从 **左侧墙、右侧墙、上方墙** 随机选择一个位置生成弹球
- 弹球触碰墙壁或球拍时发生反弹
- 每次弹球与墙壁碰撞时，玩家获得相应分数
- 若弹球在作用时长耗尽前坠落至球拍下方，则扣除该弹球的点数
- 存在一个弹球生成的高峰期。

### 游戏目标

在限定倒计时内，使用球拍接住弹球，追求最高总分。

---

## 3. 全局变量

| 变量名 | 类型 | 描述 |
|--------|------|------|
| `countdown` | `int` | 倒计时，表示一局游戏的总时长（秒） |
| `totalScore` | `float` | 总分，记录玩家当前累计获得的分数 |

---

## 4. 弹球属性

| 名称   | 描述                                       |
| ------ | ------------------------------------------ |
| 普通球 | 速度、得分、持续时间、体积均处于平均水平   |
| 飞快球 | 速度更快，分数更低，体积更小，持续时间更长 |
| 缓慢球 | 速度更慢，分数更高                         |
| 持续球 | 持续时间更长，体积更大                     |

| 属性名 | 类型 | 描述 |
|--------|------|------|
| `bounceCount` | `int` | 作用次数。弹球每次与墙壁发生碰撞时 +1，初次生成时计入 |
| maxCount | `int` | 最大作用次数 |
| `spawnRate` | `float` | 出现概率。该类弹球在一局游戏中的生成权重（0–1） |
| `points` | `float` | 点数。碰撞墙壁时加分的分值，掉落时扣分的分值 |

---

## 5. 游戏规则

### 得分机制

- **碰撞得分**：弹球每成功碰撞一次墙壁，获得 `points` 分
- **掉落惩罚**：弹球在 `activeDuration` 归零前坠落至球拍下方，扣除 `points` 分

### 游戏状态

| 状态 | 描述 |
|------|------|
| `READY` | 游戏就绪，等待玩家开始 |
| `PLAYING` | 游戏进行中，倒计时运行中 |
| `PAUSED` | 暂停 |
| `GAME_OVER` | 倒计时结束，显示最终得分 |

---

## 6. 物理参数

| 参数 | 描述 |
|------|------|
| 球拍宽度 | 固定像素值，支持响应式缩放 |
| 弹球半径 | 固定像素值 |
| 初始速度 | 弹球生成时的运动速度（方向由生成位置决定） |
| 反弹弹性系数 | 碰撞墙壁后的速度衰减比例 |

---

## 7. 界面交互

| 操作 | 触发方式 |
|------|----------|
| 移动球拍 | 鼠标在游戏区域内左右移动 |
| 开始游戏 | 点击「开始」按钮 |
| 暂停 / 继续 | 点击「暂停」按钮或按下 `Space` 键 |
| 重新开始 | 游戏结束后点击「再来一局」 |
